RENT ANYTHING PLATFORM

Rent Anything Platform(RAP) is a peer to peer renting platform.
RAP allows users to rent items and list their own idle items for other fellow users to rent.

Technology stack:-

Frontend:
- Vite + REACT + Typescript + Tailwind css

Backend:
- Java spring boot + MySQL

Setup & run the application:-

Frontend:
- First clone the repository using this command: git clone https://github.com/AshishChaudhary01/Rent-Anything-Platform.git
- Open the frontend folder in vscode(or IDE of your choice) and run: npm install && npm run dev
- Done.

Backend:
- Start MySQL server and create the schema 'rap'
- Open the backend folder in Intellij IDEA(or IDE of your choice) and simply run the application
